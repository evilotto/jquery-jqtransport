If you have several different things to swap in, you can just put them all in the page source, then the ic-get-from attributes pick the one you want. Often two different elements will need to be updated, but it would be nice to only use a single server request. What you can do is to store the server response in a undisplayed node - here it's "#storage" - then update the ui with local requests.
